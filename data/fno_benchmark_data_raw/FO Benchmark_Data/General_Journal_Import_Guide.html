D365 Finance - General Journal Import Guide

AR/AP Journal Import via Excel

Purpose: Import AR/AP general journals | Time: 20-30 min | Role: Accounting manager

Overview

TypeFileLines
AR JournalAR_Journal_Import_Updated.xlsx3,467
AP JournalAP_Journal_Import_Updated.xlsx3,474

Account Formats

Customers: SYNCUS-0001 to SYNCUS-1000
Vendors: SYNVEN-0001 to SYNVEN-1000

Pre-Requirements

  • Customers imported (SYNCUS)
  • Vendors imported (SYNVEN)
  • Chart of accounts configured
  • Budget control DISABLED

AR Journal Import

1Navigate: GL > Journal entries > General journals
2Create Journal: New > Name: Daily > Save
3Import: Lines > Import from Excel > Select AR_Journal_Import_Updated.xlsx
4Verify: 3,467 lines | Customer accounts SYNCUS-xxxx | Balanced
5Post: Validate > Validate > Post
✅ AR Journal posted: 3,467 transactions

AP Journal Import

1Create: New journal > Name: Daily
2Import: Lines > Import from Excel > Select AP_Journal_Import_Updated.xlsx
3Verify: 3,474 lines | Vendor accounts SYNVEN-xxxx
4Post: Validate > Post
✅ AP Journal posted: 3,474 transactions

Troubleshooting

IssueSolution
Budget exceededDisable budget control
Account not foundImport customers/vendors first
Unbalanced journalCheck debits = credits in Excel

Checklist

□ AR Journal posted (3,467 lines)
□ AP Journal posted (3,474 lines)
□ Trial balance balanced
