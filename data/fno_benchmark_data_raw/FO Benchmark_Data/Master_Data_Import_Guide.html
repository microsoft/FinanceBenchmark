D365 Finance - Master Data Import Guide

Customers & Vendors Import Instructions

📋 Purpose: Import customer and vendor master data using DMF

⏱️ Time: 30-45 minutes | 👤 Role: Data management administrator

📊 Master Data Overview

Data TypeFileRecordsEntity
CustomersD365_Customers_DMF_Final.csv1,000Customers V3
VendorsD365_Vendors_DMF_Final.csv1,000Vendors V2

🔢 Account Formats

Customers: SYNCUS-0001 to SYNCUS-1000

Vendors: SYNVEN-0001 to SYNVEN-1000

🚫 Pre-Requirements

These must exist before import:

  • Customer groups: 10, 20, 30, 80, 90
  • Vendor groups: 10, 20, 30, 40, 50
  • Payment terms: Net10, Net15, Net30, Net45, COD
  • Sales tax groups: No-Tax, EXMPT FOR, CA, TX, NY, FL, WA, IL
  • Payment methods: CHECK, ELECTRONIC, CC

👥 Customer Import

1 Create Import Project
Workspaces > Data management > Import
Project: Customer Master Data | Format: CSV
2 Add Customers V3 Entity
Add file > Entity: Customers V3 > Upload: D365_Customers_DMF_Final.csv

⚠️ CRITICAL: Field Mapping

You MUST adjust these mappings manually:

Source (CSV)→Target (D365)
CreditMax→Credit limit
CurrencyCode→SalesCurrencyCode
CustomerHoldStatus→CredManAccountStatusId
PaymentTermsId→PaymentTerms
PrimaryContactPersonFirstName→PersonFirstName
3 Fix Field Mappings
View map > For each row above, select correct target field > Save
4 Import & Verify
Import > Wait for Success > View log (1,000 records)
AR > Customers > All customers > Filter: SYNCUS

✅ Customers imported: 1,000 (SYNCUS-0001 to SYNCUS-1000)

🏢 Vendor Import

1 Create Vendor Project
Data management > Import
Project: Vendor Master Data | Format: CSV
2 Add Vendors V2
Add file > Entity: Vendors V2 > Upload: D365_Vendors_DMF_Final.csv

⚠️ CRITICAL: Vendor Field Mapping

Similar to customers, vendor field mappings may need manual adjustment:

Source (CSV)→Target (D365)
AddressState→AddressStateId
PaymentMethodName→DefaultVendorPaymentMethodName
PaymentTermsId→DefaultPaymentTermsName
PrimaryContactEmail→PrimaryEmailAddress
PrimaryContactPersonFirstName→PersonFirstName
PrimaryContactPhone→PrimaryPhoneNumber
SalesTaxGroup→SalesTaxGroupCode
VendorHoldStatus→OnHoldStatus
3 Adjust Vendor Field Mapping
View map > For each mapping in table above, select correct target field > Save
4 Import & Verify
Import > Wait for Success > View log (1,000 records)
AP > Vendors > All vendors > Filter: SYNVEN

✅ Vendors imported: 1,000 (SYNVEN-0001 to SYNVEN-1000)
